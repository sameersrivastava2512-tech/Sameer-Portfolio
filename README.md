# Task 1: AWS S3 Portfolio Website

## Project Overview
A simple responsive portfolio website hosted on AWS S3 and served securely using CloudFront (HTTPS).

## Technologies Used
- HTML
- CSS
- AWS S3
- AWS CloudFront
- GitHub

## Steps Performed

1. Created index.html and styles.css.
2. Created an AWS S3 bucket.
3. Enabled static website hosting.
4. Uploaded website files to S3.
5. Created a CloudFront distribution.
6. Configured HTTPS access.
7. Verified the live website.

## Task 2: Deploy Portfolio Website using Docker on AWS EC2

### Technologies Used
- AWS EC2
- Docker
- Ubuntu
- GitHub

### Steps Performed

1. Launched an Ubuntu EC2 instance.
2. Enabled HTTP and HTTPS traffic.
3. Connected to the instance using EC2 Instance Connect.
4. Updated system packages.
5. Installed Docker.
6. Started Docker service and verified the installation.
7. Cloned the GitHub repository.
8. Built a Docker image using the Dockerfile.
9. Ran the Docker container on port 80.
10. Accessed the website using the EC2 Public IP.
